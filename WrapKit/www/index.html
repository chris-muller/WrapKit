WrapKit Instructions

  1. "www" folder must contain a file named "index.html".
  2. All files and assets need to be located directly in the "www" folder.
  3. All URLs in the html and css need to be relative links, eg. "index.html" not "/" or "/index.html".
  4. All files and assets in the "www" folder need to be included in the Xcode project.
